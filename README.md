# In-house-testing
这款随堂测微信小程序，是教师教学、学生学习的好帮手。教师在小程序内创建测试，题型涵盖单选、多选、填空，还能添加主观题，轻松插入图片、公式。课堂上，学生扫码或在小程序内一键答题，提交后成绩即时反馈。教师能快速查看班级答题统计，分析知识点掌握情况，生成学情报告，为后续教学提供参考。对学生而言，通过即时反馈查漏补缺，巩固知识。借助小程序，提升课堂互动，优化教学效率 。
